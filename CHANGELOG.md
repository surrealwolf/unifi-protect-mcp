## [0.1.0] - 2025-12

### Added
- Initial release of UniFi Protect MCP
- 14 management tools for Protect operations
  - Camera Management with PTZ and RTSPS support
  - Motion & Environment Sensors
  - Smart Lighting Control
  - Audio Alerts and Chimes
  - Live Views Configuration
  - Advanced Camera Controls
  - Event Monitoring
  - Viewer Management
- Comprehensive API integration with UniFi Protect
- Docker containerization support
- Stdio transport for MCP protocol
